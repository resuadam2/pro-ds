# Tema 8 - Introducción a las GUI en Java y Python

En este tema se introduce el concepto de GUI (Graphical User Interface) y se presentan ejemplos de cómo crear interfaces gráficas en Java y Python. Se utilizan las bibliotecas Swing y JavaFX para Java y Tkinter para Python.
Se presentan ejemplos de cómo crear ventanas, botones y etiquetas en ambas bibliotecas. También se discuten las diferencias entre las dos bibliotecas y se proporcionan recursos adicionales para aprender más sobre la creación de GUI en Java y Python.
