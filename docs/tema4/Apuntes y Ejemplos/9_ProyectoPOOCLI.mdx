# Ejemplo tienda de videojuegos

En este ejemplo vamos a crear una aplicación de consola que simula una tienda de videojuegos. 

La aplicación permitirá al usuario realizar las siguientes operaciones:

- Ver el catálogo de videojuegos disponibles.
- Añadir un videojuego a la lista.
- Eliminar un videojuego de la lista.
- Salir de la aplicación.

Para ello, crearemos una clase `Videojuego` que representará un videojuego con los siguientes atributos:

- `titulo`: el nombre del videojuego.
- `plataforma`: la plataforma en la que se puede jugar.
- `precio`: el precio del videojuego.

:::warning[Nota]
Vamos a tratar la plataforma como un String de momento, pero lo ideal sería que fuese un enumerado.

De esta forma, si lo fuese, podríamos tener un método que nos devolviese todas las plataformas disponibles, y así poder seleccionar una de ellas.
:::
