# Programación

En esta página encontraréis todo el material que iremos viendo durante el curso. 

Este sitio servirá básicamente de guía, con apuntes, enunciados de ejercicios y soluciones de los mismos, con proyectos de ejemplo y ejercicios que iremos viendo en clase, o pequeños tutoriales y cursos para ir aprendiendo diferentes conceptos.

Comenzaremos programando en Python para aprender las bases mientras vamos introduciendo alguna diferencia y similutud con Java, lenguaje con el que terminaremos el curso y al que nos pasaremos en cuanto introduzcamos la programación orientada a objetos o POO.
